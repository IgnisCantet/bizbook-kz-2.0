## 📱 О приложении

BizBook KZ — полнофункциональный концепт мобильного приложения для ведения бухгалтерского и налогового учёта ИП и ТОО в Казахстане. Разработано на базе ТОО «NOVA COMP» (ОУР + НДС).

### Ключевые функции:

- ✅ **Все типы организаций**: ТОО, ИП, АО, КФХ, КСК/ОСИ, НКО, Самозанятые
- ✅ **Все налоговые режимы**: ОУР, СНР (упрощёнка), Патент, Самозанятые, КФХ
- ✅ **Полный пакет документов**: ЭСФ, ЭАВР, АВР, Счёт, Доверенность, Накладная, СФ, Договор
- ✅ **ЭЦП на мобильном**: eGov Mobile (QR/push), eGov Cloud, NCA Layer (ПК)
- ✅ **ИИ-ассистент**: ответы по НК РК 2026, авторасчёт налогов и ЗП
- ✅ **Интеграции**: КНП (salyk.kz), Портал ЭСФ, eGov, Казстат, Елиценз, Halyk Bank API
- ✅ **Налоговый календарь**: уведомления за 14, 7, 3 дня
- ✅ **Калькулятор ЗП**: для всех категорий (студент, инвалид, пенсионер, нерезидент)
- ✅ **Аналитика**: доходы/расходы по категориям, динамика за период
- ✅ **Синхронизация с банками**: Halyk, Kaspi, Jusan, ForteBank
- ✅ **НК РК 2026**: ОПВР 3.5%, СН 6%, вычет 30 МРП, порог НДС 43.25 млн ₸
- ✅ **PWA**: устанавливается как приложение на iOS и Android

## 🚀 Запуск и публикация на Vercel

### Шаг 1: Загрузите на GitHub

1. Создайте репозиторий на github.com
2. Загрузите ВСЕ файлы этой папки
3. Убедитесь, что структура правильная (см. ниже)

### Шаг 2: Деплой на Vercel

1. Зайдите на vercel.com → New Project
2. Выберите ваш GitHub репозиторий
3. Framework Preset: **Vite**
4. Build Command: `npm run build`
5. Output Directory: `dist`
6. Нажмите **Deploy**

### Шаг 3: Установите как приложение

- **iOS**: Safari → Поделиться → «На экран "Домой"»
- **Android**: Chrome → ⋮ → «Добавить на главный экран»
